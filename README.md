play20-social
============

Social module for Play Framework 2!

How to use ?
------------

* clone the repo
* execute 'play dist' in project-code
* add the dependency in your project

Build.scala :

	val appDependencies = Seq(
		// Add your project dependencies here,
	        "play20-social" % "play20-social_2.9.1" % "1.0"
    	)
